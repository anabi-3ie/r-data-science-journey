# r-data-science-journey

My personal learning lab for R, data science, visualisation, and evidence-synthesis workflows.

## What's in here

This repository is both a **portfolio** and a **practice space**. Each folder corresponds to a phase of a structured course I'm working through. Sessions are modular (30 min – 2 hr each), and every session ends with a commit.

## My background

I work in evidence synthesis and manage living systematic reviews. This learning journey is calibrated to that work — examples lean toward effect sizes, screening logs, extraction tables, and meta-analytic outputs.

## How I'm learning

- Short modular sessions
- Mix of "concept + commented code + try it" and "challenge-first"
- Every session committed with a meaningful message
- AI-assisted reflection where useful (Claude / Copilot / Positron Assistant)
- A parallel stats & meta-analysis course (kept in a separate space, but cross-referenced)
